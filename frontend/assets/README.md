# Assets Directory

Place the following image files in this directory:

## Required Assets

1. **icon.png** (1024x1024)
   - App icon (square)
   - Used for iOS, Android, and web

2. **splash.png** (1284x2778 recommended)
   - Splash screen image
   - Should have a transparent or solid background matching the app theme (#8B7355)

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon
   - Should have transparent background

4. **favicon.png** (48x48 or 192x192)
   - Web favicon
   - Small icon for browser tabs

## Color Scheme

- Primary: #8B7355 (Brown)
- Background: #F5F1EA (Warm beige)
- Accent: #A8C09A (Sage green)

## Design Notes

- Keep icons simple and recognizable
- Use warm, earthy tones
- Consider horse/rider imagery
- Ensure icons work at small sizes
- Test on both light and dark backgrounds

You can create placeholder images using any image editor or use online icon generators.
